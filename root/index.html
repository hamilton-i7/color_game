The RGB (red, green, blue) system ranges from 0 to 255. It specifies how much red, green and blue a color has.

The higher the number, the lighter the color is going to be.

For example, rgb(255, 255, 255) will have the color white as a result.

... and rgb(0, 0, 0) will have the color black as a result.

On the other hand, rgb(0, 150, 0) will result in a darker green.
